\abstract{This library introduces code-generation methods for randomized algorithms modelled
using probability mass functions. The first strategy works using a syntactic translation using
customized serialization with a minimal trusted base. The second strategy relies on modelling the
executable randomized algorithm, parameterized by the random source, directly in Isabelle and
works without customized serialization.}
